## ビルド方法

### 必要環境

- .NET SDK 6 RC1 以上

## Toolsについて

### TopoJsonConverter

TopoJsonを簡略化するものです。  
単一ポリゴンのものしか対応していません。

### TrTimeTableConverter

走時表を変換するものです。

### ObservationPointEditor

観測点を編集するやつです。  
未完成ですが、利用は可能です。
